## Double-Click to Run!

### ‚úÖ START_SmartRisk.bat
**What it does:**
- Starts the backend server (Python)
- Starts the frontend server (React)
- Automatically opens your browser to http://localhost:3000
- Keeps two command windows open (don't close them!)

**To use:**
1. Double-click `START_SmartRisk.bat`
2. Wait ~10 seconds
3. Browser opens automatically
4. Start analyzing portfolios!

---

### üõë STOP_SmartRisk.bat
**What it does:**
- Stops all backend servers
- Stops all frontend servers
- Closes everything cleanly

**To use:**
1. Double-click `STOP_SmartRisk.bat`
2. All servers stop
3. Done!

## ‚ö†Ô∏è Troubleshooting

### "Python is not recognized"
- Make sure Python is installed
- Restart your computer after installing Python

### "npm is not recognized"
- Make sure Node.js is installed
- Restart your computer after installing Node.js

### Ports already in use
- Run `STOP_SmartRisk.bat` first
- Then try `START_SmartRisk.bat` again

### Browser doesn't open automatically
- Manually open browser to: http://localhost:3000
