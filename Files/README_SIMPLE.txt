===============================================
    SMARTRISK LITE - PORTFOLIO ANALYZER
===============================================

SUPER SIMPLE INSTRUCTIONS:

1. FIRST TIME ONLY:
   - Double-click: MAKE_SHORTCUT.bat
   - This creates a nice icon to click

2. TO START THE APP:
   - Double-click: "SmartRisk Lite.lnk" 
   - (or just use START_SmartRisk.bat)
   - Browser opens automatically
   - Two black windows appear (leave them open!)

3. USE THE APP:
   - Click "Load Example Portfolio"
   - Click "Analyze Portfolio"
   - See your results!

4. TO STOP:
   - Double-click: STOP_SmartRisk.bat
   - Or just close the two black windows

===============================================

WHAT IT DOES:
- Analyzes stock portfolios
- Shows risk vs return
- Calculates Sharpe ratios
- Projects investment returns

HOW TO DEMO FOR GORDON:
1. Double-click the shortcut
2. Wait 10 seconds for browser to open
3. Load example (AAPL, MSFT, NVDA, TSLA)
4. Click Analyze
5. Show him the chart and metrics!

===============================================

TROUBLESHOOTING:

Problem: "Python not found"
Fix: Install Python from python.org

Problem: "npm not found"  
Fix: Install Node.js from nodejs.org

Problem: Browser doesn't open
Fix: Manually go to http://localhost:3000

Problem: Something is broken
Fix: 1. Run STOP_SmartRisk.bat
     2. Wait 5 seconds
     3. Run START_SmartRisk.bat again

===============================================

That's it! Keep it simple.

Questions? Check HOW_TO_RUN.md for details.
